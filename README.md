# Project-2

Hotel Booking Demand Analysis and Dynamic Pricing & Cancellation Prediction
This repository contains a collaborative project focused on the travel and hospitality sector, where revenue leakage often occurs due to unoptimized pricing and unpredictable customer cancellations. Using the Hotel Booking Demand Dataset (Kaggle), our team performs Exploratory Data Analysis (EDA) and builds predictive models to uncover the primary drivers of cancellations (churn) and explore seasonal demand trends.

Objectives
- Identify key factors influencing booking cancellations (lead time, deposit type, customer segment).
- Visualize booking curves and seasonal demand fluctuations.
- Analyze pricing elasticity and simulate dynamic pricing strategies.
- Provide actionable insights to optimize Revenue Per Available Room (RevPAR) and reduce cancellation rates.

Key Features
- Clean and reproducible Jupyter Notebook workflows.
- Customer segmentation and churn analysis.
- Seasonal demand visualization and booking curve plots.
- Baseline predictive modeling (logistic regression, decision trees).
- Business-focused KPIs: ADR (Average Daily Rate), RevPAR.

Team Collaboration
This project is developed jointly by a 5-member team. Each contributor focuses on a specific module (EDA, feature engineering, modeling, visualization, business insights) and integrates work through Git version control for seamless collaboration.
